# East-Py
 EAST: An Efficient and Accurate Scene Text Detector

## Required Software
 * **Sublime Text [Optional]**

## Required Dependencies
 * TensorFlow
 * OpenCV

## Compile 
```sh
$ cd folder
$ python test.py --east text_detection.pb --image images/exam11.png --padding 0.05
```
